## 📌 About

A front-end landing page built for a moving van service business. The site presents the company's services, showcases happy customers, and provides a clear call-to-action — all in a mobile-friendly, visually appealing layout built with **HTML5** and **CSS3**.

The project also includes an **SMS notification script** (`sms.py`) for customer communication.

---

## ✨ Features

| Feature | Details |
|---|---|
| **Responsive Design** | Fully adapts to desktop, tablet, and mobile screens |
| **Custom SVG Icons** | Hand-picked icons for services — truck, briefcase, box, bus, chat, and navigation |
| **Image Gallery** | Real photos showcasing customers — couples, families, pets welcome |
| **Clean Layout** | Organized sections with clear typography and spacing |
| **SMS Script** | Python script for sending SMS notifications to customers |

---

## 📄 Sections

- **Hero / Landing** — Company introduction with a strong headline and call-to-action
- **Services** — What the company offers, illustrated with SVG icons (packing, transport, delivery, support)
- **Testimonials / Gallery** — Customer photos (couple, family, dog) building trust and warmth
- **Contact / CTA** — Easy way to get in touch or request a quote

## ⚙ Installation

**1. Clone the repository**
```bash
git clone https://github.com/<your-username>/moving-van-service.git
cd moving-van-service
```

**2. Open in browser**
```bash
open index.html          # macOS
start index.html         # Windows
```

Or use **Live Server** in VS Code for auto-reload.
